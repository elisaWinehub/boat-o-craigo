# Boat O'Craigo Cursor Rules

Before editing any Shopify theme file:

1. Read `docs/BOAT-O-CRAIGO-DESIGN-SYSTEM.md`.
2. Reuse existing `boc-*` sections, snippets, CSS classes and design tokens.
3. Do not redesign the global header or footer unless explicitly requested.
4. Do not use `#F0F5ED`.
5. Do not invent new colours, fonts, shadows, border radii or animations.
6. Product collection images must use white backgrounds, no padding, no bottle shadow and no hover animation.
7. Product card Add to Cart buttons must use `#212121`.
8. Use Shopify sections and blocks for all merchant-editable content.
9. Use section settings for content, spacing, images, buttons and backgrounds.
10. Keep page templates modular and AI-readable.
11. Run `shopify theme check` before reporting completion.
12. Report any design-system deviation before implementing it.
